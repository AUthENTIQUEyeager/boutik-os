Installation

Prêt en moins d'une minute

BoutiK s'installe directement depuis votre navigateur. Choisissez votre appareil ci-dessous.

Android

  1. 1Ouvrez BoutiK dans votre navigateur Chrome.
  2. 2Appuyez sur les 3 points en haut à droite.
  3. 3Choisissez "Ajouter à l'écran d'accueil".
  4. 4Confirmez en appuyant sur Ajouter.

iPhone / iPad

  1. 1Ouvrez BoutiK dans Safari.
  2. 2Appuyez sur l'icône Partager (carré avec flèche).
  3. 3Choisissez "Ajouter à l'écran d'accueil".
  4. 4Confirmez en appuyant sur Ajouter.

Ordinateur

  1. 1Ouvrez BoutiK dans Chrome ou Edge.
  2. 2Cliquez sur l'icône Installer dans la barre d'adresse, ou les 3 points.
  3. 3Cliquez sur "Installer" dans le menu.
  4. 4BoutiK s'ouvre comme une application native.
